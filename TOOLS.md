# Перечень инструментов разработки (по п. 4.4 ТЗ)

Перечень программных продуктов, фреймворков и сервисов, использованных при разработке элементов 1–3 (сайт, интернет-баннер, наружный баннер).

## 1. Дизайн и фирменный стиль

| Инструмент | Назначение |
| --- | --- |
| **Pencil** | Векторный редактор для разработки макетов — лендинг (`design/landing.pen`) и баннеры (`design/banners.pen`) |
| **Inter** (Google Fonts) | Гарнитура фирменного стиля — заголовки, текст, логотип |
| **Палитра** | `#FF6B35` (Brand Orange), `#FF3D00` (Brand Deep), `#FFE2D1` (Brand Soft), `#101828` (Ink), `#FFFFFF` |

## 2. Разработка лендинга

| Инструмент | Версия | Назначение |
| --- | --- | --- |
| **React** | 19.2 | Библиотека построения интерфейса |
| **TypeScript** | 5.9 | Статическая типизация |
| **Vite** | 8.0 | Сборщик и dev-сервер |
| **Tailwind CSS** | 3.4 | Utility-first CSS-фреймворк |
| **PostCSS** + **Autoprefixer** | — | Постобработка CSS |
| **ESLint** | 9 | Линтер JavaScript/TypeScript |
| **npm** | — | Менеджер пакетов |
| **Node.js** | 20 LTS | Среда выполнения для сборки |

## 3. Деплой и хостинг

| Инструмент | Назначение |
| --- | --- |
| **Netlify** | Хостинг готового статического сайта, CDN, SSL |
| **netlify.toml** | Конфигурация сборки и заголовков (HSTS, Cache-Control) |
| **Git** + **GitHub** | Контроль версий, удалённый репозиторий https://github.com/kwtpub/Sonya-web-school |

## 4. Баннеры (рекламные носители)

| Носитель | Размер | Файл |
| --- | --- | --- |
| Интернет-баннер (соцсети, контекст) | 1200 × 628 px | `banners/web-banner-1200x628.png` |
| Наружный баннер (билборд) | 3000 × 1500 px (3 × 1.5 м, 1:0.5) | `banners/outdoor-banner-3000x1500.png` |

Источники макетов в `.pen` — `design/banners.pen`.

- **Google Fonts** — Inter
- **Браузер Chrome DevTools** — отладка вёрстки и проверка адаптивности
